拟古


青松劲挺姿，凌霄耻屈盘。种种出枝叶，牵连上松端。
秋花起绛烟，旖旎云锦殷。不羞不自立，舒光射丸丸。
柏见吐子效，鹤疑缩颈还。青松本无华，安得保岁寒。

龟鹤年寿齐，羽介所托殊。种种是灵物，相得忘形躯。
鹤有冲霄心，龟厌曳尾居。以竹两附口，相将上云衢。
报汝慎勿语，一语堕泥涂。

青松刚劲地挺着它的身姿，凌霄花攀援在松树上。 枝叶杂生，花朵的枝条倚在青松上，渐渐地爬上树顶。 秋天的花儿如红色烟霞般绽放，美丽得像云锦一般。 凌霄花对自己依赖别人的行为不感到惭愧，反而因自己遮掩住了巍峨挺拔的松树感到得意。 柏树见此，吐出了柏籽效仿松树的样子；仙鹤看到了，怀疑这不是松树，也缩起了自己的颈项。 青松却对此不以为意：它不追求外表，长得朴实无华。青松经冬不凋；即使凌霄花枝招展，它却不到冬天便会枯萎。 龟与鹤都是长寿的动物，但羽毛和龟壳是不同的。 它们都是灵物和寿者，因此而能相伴相行，忘却了自己形状的不同。 鹤有冲天的气概，龟厌烦地面上摇头摆尾的生活。 两只鹤分别用嘴衔住竹枝的一端，让龟咬着竹子中间的部分，就这样他们将龟带上了天。 飞在空中，鹤嘱咐龟说：“别说话，说话就有可能掉下去堕入泥潭。”




1.作品内容中的两点为重复符号，代表前一个字。 参考“种种出枝叶”中的“种种”，“舒光射丸丸”中的“丸丸”，和“种种是灵物”中的种种。
2.“拟古”是指诗文模仿古人风格形式的创作手法， 即这首诗是诗人参考古人风格创作的。

吴江垂虹亭作


断云一片洞庭帆，玉破鲈鱼金破柑。好作新诗继桑苎，垂虹秋色满东南。
泛泛五湖霜气清，漫漫不辨水天形。何须织女支机石，且戏常娥称客星。
时为湖州之行

断云般的小船漂浮在太湖上。湖中的鲈鱼白润如玉，柑桔亮如金。 我真想接踵陆羽，卜居洞庭，好时时写出新诗。眼前垂虹亭畔的秋色弥漫大地。
广阔无边的湖面上充满了霜天的清气，水天一色难辨分界。 何必跋山涉水去寻找织女的支机石？且在这儿游玩嬉戏，作一回嫦娥的客人。




1.帖中“霜”字为错字，右边注一小字“金”为正确内容，此为一种修改方式。
2.此诗为诗人在湖州之行时游览太湖所作，描绘洞庭之美景。前四句被厉鹗收录在《宋诗纪事》中。
3.“洞庭”和“五湖”皆为太湖别称；今“五湖”称洞庭湖，鄱阳湖，太湖，洪泽湖，和巢湖（古时太湖有别称“洞庭”，而洞庭湖有别称“云梦”）。
4.诗中的松江四腮鲈和洞庭红蜜桔为太湖时令美味。
5.“桑苎”代指茶圣陆羽。陆羽号桑苎翁，曾隐居湖州，生前以文人著称且“词艺卓异“。亦有说“桑苎”一词代指故乡。
6.支机石典故，出自：“昔有一人寻河源，见妇人浣纱，以问之，曰：‘此天河也。’乃与一石而归。问严君平，云：‘此支机石也。’”
7.客星在古代指天上新出现的星。常与“客星犯帝座”的典故联系，特指东汉隐士严光。亦有解释称”客星“为”客人“。

入境寄   集贤林舍人


扬帆载月远相过，佳气葱葱听颂歌。路不拾遗知政肃，野多滞穗是时和。
天分秋暑资吟兴，晴献溪山入醉哦。便捉蟾蜍共研墨，彩笺书尽剪江波。

披星戴月乘船扬帆经过这里，远远地望见佳气盈盈，听闻颂歌阵阵。 这里路不拾遗，可知治理严肃；野外还有富余滞留的谷穗，可见时局之和。 天有秋凉暑气供诗人吟咏，晴有美丽溪山令人沉醉。 于是我捉来蟾蜍研磨，在彩笺上书写完这诗句，我便将乘风破浪而来。




1.此诗为诗人受林舍人林希（时任湖州知府）之邀前往湖州游玩，通报他已经进入湖州之境所作。
2.佳气为祥瑞之气，喻湖州兴盛，治理有方。
3.“哦”字义“吟咏”。
4.“蟾蜍”指“蟾滴”，即蟾蜍形状的砚滴。砚滴为文房中用于研磨的水盂类器物，可以储水；倾斜砚滴使水滴于墨砚，在用墨条与水研磨。 因蟾蜍意象美好（“蟾宫折桂”寓仕途如意，“三足蟾”有旺财指意），宋人广泛使用蟾滴。

越窑青瓷蟾滴 现藏于慈溪市博物馆

5.笺指小而华贵的纸张，古时用于题咏或书信。

重九会郡楼


山清气爽九秋天，黄菊红茱满泛船。千里结言宁有后，群贤毕至猥居前。
杜郎闲客今焉是，谢守风流古所传。独把秋英缘底事，老来情味向诗偏。

山峦清新，空气爽朗，正值那九月的秋天时节，黄色的菊花和红色的茱萸，满满地装点在泛行于水面的船只上。 与朋友们千里之外定下的约定，又怎会有延误呢，众多贤能之士都来到了这里，我惭愧地置身在前面的位置。 像杜牧那样的闲情逸致的人，如今又在哪里呢，谢灵运太守的风流韵事，从古至今一直被人们传颂着。 独自手拿着秋天的菊花，到底是因为什么呢，原来是人到老年，对诗歌的情致和趣味变得更加偏爱了。




1.“重九”为重阳节别称，诗人在重阳与一群文人在郡楼相会。
2.“缘底”表“为什么”。

和   林公岘山之作


皎皎中天月，团团径千里。震泽乃一水，所占已过二。
娑罗即岘山，谬云形大地。地惟东吴偏。山水古佳丽。
中有皎皎人，琼衣玉为饵。位维列仙长。学与千年对。
幽操久独处。迢迢愿招类。金飔带秋威。欻逐云樯至。
朝隮舆驭飙。暮返光浮袂。云盲有风驱。蟾餮有刀利。
亭亭太阴宫。无乃瞻星气。兴深夷险一。理洞轩裳伪。
纷纷夸俗劳。坦坦忘怀易。浩浩将我行。蠢蠢须公起。

明亮的月亮高悬在天空中央，圆圆的月影仿佛径直跨越千里之遥。 震泽不过是一片水域，却占了广阔的天地。 岘山就如同天上的月宫，有人说它的形状覆盖了大地。 此地只是东吴的一处偏远之地，但山水自古以来就十分秀丽。 山中有一位品行高洁的人，身着如美玉般洁净的衣裳，如同以美玉为食般超凡脱俗。 他的地位如同位列仙班的长者，学问能与流传千年的智慧相媲美。 他有着幽雅的操守，独自隐居，却希望能与志同道合的人相聚。 秋风带着秋天的威严吹来，像金色的气流一般，忽然驱逐着云帆。 早晨太阳升起，云气随着车驾如疾风般升起；傍晚，余晖洒在衣袖上仿佛浮动的微光。 云朵盲目地被风驱赶着移动，月亮的阴影像锋利的刀刃。 高耸的月宫中，莫非是在仰望星象之气。 他兴致高雅，将险峻与平坦视为一体，道理透彻，能看穿官场服饰下的虚伪。 世俗之人纷纷夸耀自己的辛劳，而他却能坦然地忘却这些俗务。 浩浩荡荡的景象仿佛要带着我前行，而那些懵懂无知的人还需要您起身引领。




1.“震泽”为太湖别称。
2.“蟾”代指月亮，古人认为月亮上有蟾蜍居住，故以银蟾称月亮。“太阴宫”指月宫。
3.“夷”表平坦。

送王涣之彦舟


集英春殿鸣捎歇。神武天临光下澈。鸿胪初唱第一声。白面玉郎年十八。
神武乐育天下造。不使敲枰使传道。衣锦东南第一州。棘璧湖山两清照。
襄阳野老渔竿客。不爱纷华爱泉石。相逢不约约无逆。舆握古书同岸帻。
淫朋嬖党初相慕。濯发洒心求易虑。翩翩辽鹤云中侣。土苴尫鸱那一顾。
迩来器业何深至。湛湛具区无底沚。可怜一点终不易。枉驾殷勤寻漫仕。
漫仕平生四方走。多与英才并肩肘。少有俳辞能骂鬼。老学鸱夷漫存口。
一官聊具三径资。取舍殊涂莫回首。
元祐戊辰九月廿三日   溪堂米黻记

春日的集英殿中，鸣梢之声刚刚停息，神宗皇帝降临，光芒照彻下方。 大鸿胪开始唱名，第一个人就是王郎。他面色白皙，此时年仅十八岁。 神宗皇帝以他的恩泽养育天下万物，他不让王郎隐居，而是让王郎教化百姓。 王郎衣锦而来，执掌这东南第一胜地，相美玉一般的湖光山色映照着这片山川。 我这个襄阳的野老，只是垂钓隐居的闲客，我不爱荣华，却爱山水之美。 你我相逢，不必相约再见，却要道声长别。 我将乘着车马，手握古书，潇洒游历。 那些淫朋邪党起初也羡慕这自在，但要他们洗尽杂念真正归隐又迟疑顾虑。 丁令威化作偏偏云中的仙鹤，那些土芥、秽鸟怎会入他之眼。 王郎的才能学识近来更加深厚，就如深湛的太湖中没有低小的沙洲。 他的殷勤之心始终没有改变，远道而来寻我这放浪的闲人。 我平生浪荡四海为家，所幸常能与当世英才相交往。 年青时总是口出狂言骂神骂鬼，如今老来只学范蠡难得开口。 留此一官聊以供隐居的资用，你我取道不同，请君去后不必回头相顾。




1.单字右侧有点为删除符号，一般为三点。参考该图右数第七行第一个字“清”，和第十二行第七个字“業（业）”。
2.“枉驾殷勤寻漫仕”中的“殷”（yīn）字少一捺，缺写末笔，为避讳宋太祖父亲（名弘殷）。《拟古》中亦有“殷”字，但因该处读yān表颜色，故未避讳。
3.米黻为米芾原名，41岁时改“黻”为“芾”。据传是为了掩盖胡人出身，称自己为楚王后裔（楚王祖先熊泽姓芈，“芾”字与“芈”相像，又与“黻”字谐音）。
4.春季在集英殿举行殿试；鸿胪为掌朝会礼仪的官员，传唤新科进士。
5.敲枰，下棋，代指隐居。
6.东南第一州，指杭州。
7.辽鹤，辽东人丁令威学道后化鹤归辽，徘徊空中而言曰：“有鸟有鸟丁令威，去家千年今始归。”
8.具区为太湖别称。
9.鸱夷，鸱夷子皮，春秋范蠡之号。
